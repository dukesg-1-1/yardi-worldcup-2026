Set up live score sync

Scores will sync in real time for everyone with the link. This takes about 60 seconds and only needs to be done once — by whoever will be keeping score.

1
Go to jsonbin.io and sign up for a free account (takes ~30 seconds with Google).
2
Once logged in, go to API Keys in the top menu and copy your Master Key.
3
Paste it here — the page will create a shared bin and give you a link to send to the team.
